## Why Start a Blog

I've been accumulating notes in my Obsidian vault for a while. The idea of organizing and sharing them was always there, but it never made it to publishing.

Setting up this blog with Hugo + GitHub Pages, I established a few criteria.

## Platform Selection Criteria

1. **SEO**: Full control over custom domains, sitemaps, and meta tags
2. **GEO (Generative Engine Optimization)**: JSON-LD structured data for AI search engine citations
3. **AEO (Answer Engine Optimization)**: FAQ/HowTo schema markup support
4. **Ownership**: 100% ownership of markdown source via Git

## Tech Stack

- **Hugo** — Go-based static site generator
- **PaperMod** — Minimal, SEO-optimized theme
- **GitHub Pages** — Free hosting with automated deployment
- **Obsidian** — Content writing and management

I'll be writing about data engineering, AI, and productivity tools going forward.
